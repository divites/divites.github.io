Paid Riot Crates

Derived From Full-Time

This is a system where people can purchase riots either via developer product or in game currency, when a riot is purchased the default time is added and the riot goes on for the length using OS, all clients are fired to hopefully have the UI in sync. People can purchase another riot onto an existing riot to add a riot to the riot queue, if the riot purchased is the same tier riot it is added to the time, else it is added to the riot queue. After 5 minutes of a riot ending the next riot in the riot queue that was added the sooner will be the registered riot, it will also add on any other riot contributor who purchased the same riot and is also in the queue.
